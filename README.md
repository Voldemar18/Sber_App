## 📖 О проекте

**Sber for Employees** — это мобильное приложение, разработанное в рамках хакатона **"Весна 2026"** для повышения эффективности сотрудников дилерских центров.

Приложение превращает выполнение KPI в прозрачную и выгодную игру, мотивируя сотрудников увеличивать долю банка в портфеле через понятную механику статусов, финансовую выгоду и удобные инструменты планирования.

## 🔥 Основные функции

### 1️⃣ Текущий статус (Silver / Gold / Black)
- Визуальный бейдж и прогресс-бар
- Прогноз роста дохода и экономии при переходе на следующий уровень
- *"Как ускорить переход"* → переход в сценарный калькулятор

### 2️⃣ Сценарный калькулятор
- Интерактивные ползунки: изменение количества сделок, объема, доли банка
- Мгновенный пересчет: новый рейтинг, уровень, доход и экономия
- Инструмент для планирования личной эффективности

### 3️⃣ Детализация рейтинга
- Разбор баллов по трем метрикам: объем, количество сделок, доля банка
- Подсказки: *"Как рассчитывается"* и *"Как увеличить"*
- Связка с калькулятором для моделирования роста

### 4️⃣ Привилегии уровня
- Разделы: Активные и Заблокированные
- Финансовый эффект каждой привилегии в рублях
- Детальная информация и расчет следующего статуса

### 5️⃣ Личный финансовый эффект
- Агрегация всех выгод: доп. доход, экономия по ипотеке, кэшбэк, стоимость ДМС
- Главная цифра: *"Ваша общая выгода в 2026 году"*

### 6️⃣ Задачи месяца
- Персональные цели с прогресс-баром, дедлайном и наградой в баллах
- Пример: *"Увеличить долю банка до 50% → +6 баллов"*

### 7️⃣ Результаты дня
- Быстрый ввод данных: сделки, объем, доп. продукты
- Кнопка детализации для перехода к аналитике

### 8️⃣ Рейтинг
- Топ-10 внутри дилера и по региону
- Персональная позиция сотрудника

### 9️⃣ Обучение
- Короткие видеоуроки и тестирование
- Баллы за прохождение модулей

### 🔟 Поддержка
- Чат с персональным менеджером
- История обращений и горячая линия

### 1️⃣1️⃣ Профиль сотрудника
- Данные сотрудника, уровень привилегий, дата регистрации
- Авторизация через Sber ID

### 1️⃣2️⃣ Новостной блок
- Трансляция новостей, изменений тарифов и акций
